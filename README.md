# [SDF09] Responsive Footer Design with Tailwind CSS

This is the readme file for the Tailwind CSS footer project 

## Description

The purpose of the project was to create a responsive footer for a website using Tailwind CSS. The project's  aim was enhance our CSS skills, focusing on responsive design and the utility-first approach of Tailwind CSS.

## Design Choices 

Seeing as though we had the UX/UI design provided, there wasnt much design choices I needed to make.

## Features 

Overall I was quite pleased with the way the Footer  came out, I am most proud of: 

* learning how to use Tailwind and applying it to the project.
